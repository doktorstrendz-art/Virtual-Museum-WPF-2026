 Виртуальный музей WPF 2026

Десктопное приложение для каталогизации виртуальных музеев.

 🚀 Технологии
- C# / WPF
- Entity Framework
- SQL Server

 ✨ Функционал
- Каталогизация экспонатов
- Система бронирования
- Ролевая модель (Гость/Клиент/Админ)
- Гибридная оплата

 📦 Как запустить
1. Клонируйте репозиторий
2. Откройте решение в Visual Studio
3. Восстановите NuGet пакеты
4. Запустите проект
